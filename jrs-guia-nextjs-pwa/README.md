# Guia Prático JRS (Next.js, mobile-first, PWA)

Projeto Next.js (App Router) para consulta rápida de prazos, checklists e matriz RACI (DGPM-406 + Ordem Interna JRS).

## PWA (instalar no iPhone)
Após publicar (ex.: Vercel), abra a URL no Safari → **Compartilhar** → **Adicionar à Tela de Início**.
O guia funcionará como “app” e terá cache offline básico.

## Deploy recomendado (sem terminal)
1. Crie um repositório no GitHub.
2. Faça upload de todos os arquivos desta pasta via interface web.
3. Na Vercel: **New Project** → selecione o repositório → **Deploy**.

## Rodar local (opcional)
```bash
npm install
npm run dev
```
